# 🍺 AlcoholTracker - 飲酒記録アプリ

毎日のアルコール飲酒量を記録・管理するiPhoneアプリです。

## 機能

### 📝 記録機能
- 飲み物の種類・量・日時を記録
- 純アルコール量の自動計算
- 20種類以上のプリセット飲み物
- カスタム飲み物の追加

### 📅 カレンダー表示
- 月間カレンダービュー
- 日別の飲酒量インジケーター
- 休肝日の可視化

### 📊 統計・グラフ
- 週間/月間/年間の飲酒量推移
- 飲み物種類別の円グラフ
- 曜日別傾向分析

### 🎯 目標設定
- 週間純アルコール量の上限設定
- 休肝日目標の設定
- 目標超過時のアラート

### 🔔 通知機能
- 毎日のリマインダー
- 目標超過警告
- 週間サマリー通知

## 技術スタック

- **UI**: SwiftUI
- **データ**: SwiftData
- **グラフ**: Swift Charts
- **通知**: UserNotifications
- **最小OS**: iOS 17.0

## セットアップ

### Xcodeでプロジェクトを開く

1. Xcodeを開く
2. 「Create a new Xcode project」を選択
3. 「iOS」→「App」を選択
4. 以下の設定で作成:
   - Product Name: `AlcoholTracker`
   - Interface: `SwiftUI`
   - Language: `Swift`
   - Storage: `SwiftData`
5. 作成したプロジェクトにソースファイルをコピー

### ビルド要件

- Xcode 15.0以上
- iOS 17.0以上
- macOS Sonoma以上（開発環境）

## 純アルコール量の計算式

```
純アルコール量(g) = 飲酒量(ml) × アルコール度数(%) × 0.8
```

### 目安

| 飲み物 | 量 | 純アルコール量 |
|--------|-----|---------------|
| ビール (5%) | 350ml | 14g |
| 日本酒 (15%) | 180ml (1合) | 22g |
| ワイン (12%) | 125ml | 12g |
| ウイスキー (40%) | 30ml | 10g |
| チューハイ (5%) | 350ml | 14g |

## 厚生労働省ガイドライン

- 1日の適量: 純アルコール20g程度
- 週に2日は休肝日を設ける
- 女性や高齢者は上記の1/2〜2/3程度
